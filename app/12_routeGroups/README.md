# If you want to have a different layout for different pages, you can define a route group with its own layout.

To do so, add a folder with (folderName), e.g. (defaultContent) or (landingPage) and put a layout file inside each.
Each route that is placed inside the folders will take this layout.
Like this, different components can be added or left out, e.g. a header on all pages but not on the landing page

# not-found.ts files have to be placed inside each group and not on the same level as the group folders.
